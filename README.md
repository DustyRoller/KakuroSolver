# KakuroSolver
A .NET Core solver for Kakuro puzzles.

## Usage

Puzzles to be solved should be in .txt files in the following format:
<pre>
|  x  |17\  |24\  |  x  |  x  |
|  \16|  -  |  -  |20\  |  \  |
|  \23|  -  |  -  |  -  |15\  |
|  x  |  \23|  -  |  -  |  -  |
|  x  |  x  |  \14|  -  |  -  |
</pre>
To run the solver use the following command:
 
 KakuroSolver.exe [Path to puzzle file]
